Photo & Video Enhancer

This project now uses a Flask backend for real image enhancement. Please run the server and open it in your browser.

How to run

  1. Open PowerShell in this folder:
    2. cd "c:\Users\rinku\Desktop\215"
  2. Install dependencies:
    3. python -m pip install -r requirements.txt
  3. Start the app:
    4. python app.py
  4. Open the browser at:
    5. http://127.0.0.1:5000

Once the server is running, use the web page to upload many photos and download a ZIP of enhanced images.

Ready for batch photo enhancement on your local machine.
